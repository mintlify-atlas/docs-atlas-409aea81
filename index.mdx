      Quick Start
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get your Null instance running in minutes with Docker Compose
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Null repository to your local machine.
      
      ```bash
      git clone https://github.com/Fernand-O-band01/null.git
      cd null
      ```
    </Step>
    
    <Step title="Start the infrastructure">
      Launch MongoDB, Kafka, PostgreSQL, and MailDev using Docker Compose.
      
      ```bash
      docker-compose up -d
      ```
      
      <Info>This will start all required services: MongoDB (port 27017), Kafka (port 9092), PostgreSQL (port 5431), PgAdmin (port 5050), and MailDev (port 1080).</Info>
    </Step>
    
    <Step title="Start the backend">
      Navigate to the backend directory and run the Spring Boot application.
      
      ```bash
      cd null_services_backend
      ./mvnw spring-boot:run
      ```
      
      The API will be available at `http://localhost:8088`
    </Step>
    
    <Step title="Start the frontend">
      In a new terminal, navigate to the frontend directory and start the Angular app.
      
      ```bash
      cd null_services_frontend
      npm install
      npm start
      ```
      
      Open your browser to `http://localhost:4200` to access the application.
    </Step>
  </Steps>
</div>
