## What is Null?

Null is a full-featured communication platform inspired by Discord, built from scratch with modern architecture and best practices. It enables real-time messaging, server and channel management, friend connections, and secure authentication.

Designed as a learning project to demonstrate scalable, event-driven architecture, Null showcases how to build production-ready communication systems using industry-standard technologies.

## Technology Stack

Null is built with a modern, production-ready technology stack:

### Backend
- **Spring Boot 3.4.2** - REST API framework
- **Java 17** - Programming language
- **Spring Security** - JWT-based authentication and authorization
- **Spring Data JPA** - PostgreSQL integration for user data
- **Spring Data MongoDB** - NoSQL database for chat messages
- **Spring Kafka** - Event streaming platform
- **Spring WebSocket** - Real-time communication with STOMP over SockJS
- **Thymeleaf** - HTML email template engine
- **Lombok** - Code generation and boilerplate reduction

### Frontend
- **Angular 21** - Single Page Application framework
- **TypeScript 5.9** - Type-safe JavaScript
- **RxJS** - Reactive programming for async operations
- **SockJS & STOMP** - WebSocket client libraries
- **Bootstrap 5** - UI component library
- **TailwindCSS** - Utility-first CSS framework

### Infrastructure
- **Docker & Docker Compose** - Container orchestration
- **PostgreSQL** - Relational database for user accounts and authentication
- **MongoDB** - Document database for chat messages and conversations
- **Apache Kafka** - Message broker for event streaming
- **MailDev** - SMTP server for testing email notifications
- **PgAdmin** - PostgreSQL administration tool

## Architecture Highlights

<Note>
  Null uses a **monorepo structure** with separate frontend and backend projects, orchestrated by Docker Compose for seamless development.
</Note>

- **RESTful API** - Spring Boot backend exposes endpoints at `/api/v1`
- **Real-time Communication** - WebSocket connections for instant messaging
- **Event Sourcing** - Kafka topics handle asynchronous email delivery
- **Dual Database Strategy** - PostgreSQL for relational data, MongoDB for chat messages
- **Stateless Authentication** - JWT tokens enable scalable session management

## Use Cases

Null is ideal for:

- **Learning Modern Architecture** - Understand how to build scalable, event-driven systems
- **Portfolio Projects** - Demonstrate full-stack development skills
- **Team Communication** - Deploy as an internal chat platform
- **Microservices Study** - Learn Kafka-based async communication patterns
